# 🚀 Webアプリケーション基礎研修

このリポジトリは研修用のテンプレートです。  
GitHub Codespaces を使って、ブラウザだけで開発環境が立ち上がります。

## はじめかた

1. 右上の緑色の **「Use this template」** ボタンをクリック
2. **「Create a new repository」** を選択
3. Repository name に好きな名前を入力（例: `my-training`）
4. **「Create repository」** をクリック
5. 作成されたリポジトリで **「Code」→「Codespaces」タブ →「Create codespace on main」** をクリック
6. 数十秒待つと VS Code が起動します 🎉

## フォルダ構成

```
phase1/   ← Phase 1: HTMLだけで作る
phase2/   ← Phase 2: APIサーバーを作る
phase3/   ← Phase 3: DBを追加する
phase4/   ← Phase 4: フロントとバックの結合
```

各Phaseで作成したファイルを対応するフォルダに保存してください。

## 使い方のヒント

- **Claudeで生成したコード** → Codespaces のエディタにコピー＆ペースト
- **ターミナル** → 画面下部の「Terminal」タブで `node server.js` などを実行
- **プレビュー** → ポートが自動転送されるので、表示されるURLをクリック
- **保存** → Ctrl+S（Mac: Cmd+S）で自動保存されます

## 困ったら

- Codespaces が固まったら → ブラウザをリロード
- ターミナルが見つからない → メニュー「Terminal」→「New Terminal」
- ポートが開かない → 左サイドバーの「Ports」タブを確認
